# My Neovim Setup

bat: `brew install bat`  
lua-language-server: `brew install lua-language-server`

Keymaps

`gcc` - line comment  
`gbc` - block comment

`<leader>t` - toggle neo-tree

`<Space>*` - Find references  
`<Space>&` - Implementation  
`C-]` - Definition
